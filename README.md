# Hekili-Titan

## 插件介绍

Hekili-Titan 是基于 SimulationCraft 行动列表的魔兽世界技能优先级提示助手，专门为泰坦时光服版本优化。该插件能够实时分析战斗状态，为玩家提供最优的技能使用建议，帮助提升DPS输出和坦克效率。

**版本信息：** v3.80.0-2.0.12-泰坦特别版  
**支持版本：** 泰坦时光服  
**作者：** Hekili, Hutu, 风雪, 哑吡Wind

## 主要功能

### 🎯 智能技能优先级
- 基于 SimulationCraft 算法，提供最优技能使用顺序
- 实时分析战斗状态，动态调整技能优先级
- 支持单目标和多目标战斗场景

### 🛡️ 多职业支持
- **死亡骑士：** 鲜血（坦克/输出）、冰霜、邪恶
- **德鲁伊：** 平衡、野性（猫/熊）、恢复
- **猎人：** 野兽控制、射击、生存
- **法师：** 奥术、火焰、冰霜
- **圣骑士：** 防护、惩戒、神圣
- **牧师：** 戒律、神圣、暗影
- **潜行者：** 刺杀、战斗、敏锐
- **萨满祭司：** 元素、增强、恢复
- **术士：** 痛苦、恶魔学识、毁灭
- **战士：** 武器、狂怒、防护

### ⚡ 实时战斗监控
- 技能冷却时间监控
- 资源管理（符文、能量、法力等）
- 增益/减益效果追踪
- 目标状态分析

### 🎮 用户友好界面
- 可自定义显示位置和大小
- 支持多种UI皮肤和主题
- 拖拽式界面调整
- 多配置文件支持

## 安装教程

### 方法一：手动安装
1. 下载最新的插件压缩包
2. 解压到魔兽世界插件目录：`World of Warcraft/_classic_/Interface/AddOns/`
3. 确保解压后的文件夹名为 `Hekili`
4. 启动游戏，在角色选择界面启用插件

### 方法二：插件管理器安装
1. 使用 CurseForge、WowUp 或其他插件管理器
2. 搜索 "Hekili" 并选择 Wrath 版本
3. 安装并启用插件

## 使用说明

### 基本使用
1. 进入游戏后，插件会自动加载
2. 在屏幕中央会出现技能提示框
3. 按照提示框中的技能顺序使用技能
4. 插件会根据战斗情况自动调整优先级

### 配置选项
在游戏中输入 `/hekili` 或点击小地图图标打开配置界面：

**显示设置：**
- 调整提示框位置和大小
- 设置透明度
- 选择显示模式（简单/详细）

**战斗设置：**
- 启用/禁用特定专精
- 设置AOE技能阈值
- 配置PVP模式

**高级选项：**
- 导入/导出配置
- 重置默认设置
- 调试模式

### 快捷键
- **Alt + 左键拖动：** 移动提示框
- **Alt + 右键：** 锁定/解锁位置
- **/hekili toggle：** 显示/隐藏插件
- **/hekili config：** 打开配置界面

## 技术支持

### 常见问题

**Q: 插件不显示技能提示？**  
A: 检查插件是否启用，尝试重新加载界面（/reload）

**Q: 技能优先级不正确？**  
A: 确保选择了正确的专精，检查配置中的战斗设置

**Q: 与其他插件冲突？**  
A: 尝试禁用其他战斗相关插件，或调整Hekili的显示位置
